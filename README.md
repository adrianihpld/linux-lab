# 🐧 Linux Lab

A hands on lab environment designed to showcase practical Linux system administration, automation, containerization, and orchestration skills. This repository contains real-world examples using tools like Ansible, Podman, and Kubernetes with a focus on Red Hat based systems.

## What This Project Covers

- 🛠️ Infrastructure as Code with **Ansible**
- 🧱 Containerization using **Podman**
- ☸️ Kubernetes manifests and orchestration (via **Minikube**)
- 🐚 Bash scripting for automation and monitoring
- 🔒 SELinux and firewalld configuration
- ⚙️ `systemd` unit files for service management
- ⚡ Terminal dotfiles for productivity

**Why I Built This**
I created this lab to develop and demonstrate practical Linux skills in a reproducible and structured way. It reflects my learning journey and serves as a portfolio to showcase my experience with:

- Managing Linux systems
- Automating workflows with Ansible
- Running containers securely
- Orchestrating apps with Kubernetes

**Skills Demonstrated**
- Linux CLI & shell scripting
- Configuration management
- Container lifecycle management
- SELinux & firewall security
- Systemd service handling
- YAML and Kubernetes concepts
- Git workflow and repo documentation
